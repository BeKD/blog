# 基于SpringBoot的入门级个人博客平台


本项目基于b站李仁密老师的个人博客项目改造而成，将底层用mybatis重构，适合新手入门练习

## 技术架构：

SpringBoot + Mybatis + thymeleaf，semanticUI
